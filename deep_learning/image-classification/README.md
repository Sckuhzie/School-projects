# Image classification

TD 1 by Paul Lacroix

## Installation

The python version used for this code is python 3.13.0

To run the code, run

```bash
pip install -r requirements.txt
```

## Description

The files `knn.py`, `mlp.py` and `read_cifar.py` contain the main functions to run the program. An example of how there are used are presented in the files `plot_result_knn.py` and `plot_result_mlp.py`

The plot results are in the `results` folder. They have been generated using the code in `plot_result_knn.py` and `plot_result_mlp.py`
